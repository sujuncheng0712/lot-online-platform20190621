# Agent/Dealer 代理商/经销商
##API
```
/api/agents
/api/dealers
```
##字段
```
id              -> ID
uuid            -> 商家编号(aid/did)
state           -> 账号状态
type            -> [保留字段]
```
##基础信息
```
oranization     -> 单位名称
abbreviation    -> 单位简称
area            -> 所在地(地区)
address         -> 所在地(详情)
logo            -> Logo 图像
domain          -> 单位网站
contact         -> 联系人
mobile          -> 手机号
wechat          -> 微信号
email           -> 邮箱
```
##账户信息
```
username        -> 用户名(唯一[手机号])
password        -> 密码(MD5+Salt)
region          -> 所属区域
superior        -> 所属上级
```
##隐藏信息
```
created_at      -> 创建时间
update_at       -> 修改时间
vid             -> # Vendor
remark          -> #
```
